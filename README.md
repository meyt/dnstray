# DNSTray

Set DNS from Tray icon.

![dnstray-screenshot](https://github.com/meyt/dnstray/assets/10482278/8d96d533-2eeb-4bf6-8ae6-8ad06b283ae9)

## Usage

### Windows 10, 11

1. Download the binary:
   [x64](https://github.com/meyt/dnstray/releases/latest/download/dnstray.exe)
   | [arm](https://github.com/meyt/dnstray/releases/latest/download/dnstray-arm.exe)
   | [arm64](https://github.com/meyt/dnstray/releases/latest/download/dnstray-arm64.exe)
2. Run as administrator

### Linux

1. `wget -O dnstray https://github.com/meyt/dnstray/releases/latest/download/dnstray-linux-amd64`
2. `chmod +x dnstray`
3. `./dnstray`

### Mac OSX

Not yet...

## Configuration

After first run, the `config.toml` file will create automatically.
You can change DNS servers list by editing that file and restart the app.
